# ReRa1n
### Instructions for Linux:
1. git clone https://github.com/aidangamzer/rera1n.git
2. cd rera1n
3. sudo cp rera1n.sh /usr/bin
4. sudo chmod +x /usr/bin/rera1n.sh
5. sudo apt-get update --fix-missing
6. rera1n.sh
